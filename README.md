# hackd

A Hacker News React Native powered iOS application. Built to learn React Native.

Currently a work in progress.

### Run Locally

```bash
# Clone hackd
$ git clone https://github.com/lukakerr/hackd.git

# Change directories
$ cd hackd

# Install dependencies
$ npm install

# Run the app
$ react-native run-ios
```
